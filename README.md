# Wishlistz Representative Website

A professional representative website for Wishlistz, designed to showcase team transparency, provide quick access to dashboards, and integrate a responsive chatbot for customer engagement.  
This project emphasizes modular frontend architecture, clean UI/UX, and maintainable code for future expansion.

---

## ✨ Features
- **Dashboard Layout**
  - Transparent team representatives card
  - Statistics and quick enquiry sections side by side
  - Modular sidebar with reusable context
- **Chatbot Integration**
  - Floating minimized button
  - Expandable chat window with dark/light mode
  - File attachment support
  - Sign Out popup confirmation
- **Responsive Design**
  - Works seamlessly across desktop, tablet, and mobile
- **Modern UI Enhancements**
  - Gradient buttons, color‑coded status badges
  - Hover effects for cards
  - Popup modals for actions
